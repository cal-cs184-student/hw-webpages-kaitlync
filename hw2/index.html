Part 6: Loop Subdivision for Mesh Upsampling

In this section we want to convert coarse polygon meshes into higher resolution ones for better simulation, rendering, and display. To do so we need an upsampling algorithm that interpolates well and approximates the original mesh data. We need an algorithm that allows us to do this in 3D, since 2D bilinear filtering does not translate well into 3D. In this section we will be implementing the loop subdivision algorithm to upsample meshes.

Loop subdivision upsamples by essentially:
1. 4-1 subdivision: subdividing each triangle into 4 smaller triangles and then
2. updating the vertices of this subdivided mesh based on a weighting scheme
We can apply 4-1 subdivision over all triangles of a mesh by first splitting every existing/”old” edge to create “new” vertices, then edge flip every edge that connects a “new” vertex and an old one. Then we update each vertex of this subdivided mesh by using a weighted average of neighboring vertex positions.
