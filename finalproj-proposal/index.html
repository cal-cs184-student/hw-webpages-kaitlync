Summary:

In this project, we aim to simulate realistic fluid motion using core ideas of fluid dynamics and physics. We will extend the project to simulate varying fluids, like honey and possibly lava, and simulate their interaction with environments and objects.

Problem Description

Simulating fluids realistically in 3D is a challenging task, and is useful in many applications like visual effects, engineering, and environmental modeling. In our project, we aim to tackle the problem of realistically simulating a fluid. This can be done by treating the fluid as many particles, or alternatively, as a continuum whose time evolution is governed by the Navier-Stokes equation. However, using a PDE solver for the Navier-Stokes equation can be extremely slow. In our project, we will first use a Navier-Stokes solver, and then attempt to speed up the simulation by using an alternative to the Navier-Stokes technique: the Lattice Boltzmann Method. We will also experiment with making different types of fluids, like honey and lava, and try to make them look realistic.
